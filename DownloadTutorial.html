how to install flappy veggies for windows

1. download the .zip file by clicking the download button below
2. right-click the downloaded .zip file (usually in your downloads folder)
3. click on the "Extract all..." button and extract the files to a new folder
4. open the new folder to find the program, it should now have a logo
6. You can now run the program!

If you want to find the program easier, you can put a shortcut on your desktop by right-clicking
the program and selecting "Create shortcut" then dragging the shortcut to your desktop.
